# Contributing to Awesome Multimodal Extraction

Thanks for helping keep this the most current multimodal-extraction list on GitHub.

## The rules

1. **It must be real and maintained.** Link to a live project, model, or dataset.
   Dead links and abandoned repos get rejected. CI runs a link check on every PR.
2. **One line, factual.** Format: `` - [Name](https://link) - What it does. ``
3. **Right section.** Put it where a reader would look for it.
4. **Stay on topic.** This list is about extracting entities and structured data
   from images + text (VLMs, document AI, OCR-to-structured, multimodal NER).
   Pure text-only tools belong in the sibling
   [Awesome Entity Extraction](https://github.com/shivamnegi92/awesome-entity-extraction).

## How to add an entry

1. Fork and edit `README.md`.
2. Add your one-line entry under the right heading.
3. Run the link check if you can: `python3 scripts/check-links.py`.
4. Open a PR and say briefly why it belongs.

Quality over quantity.
