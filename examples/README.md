# Runnable Examples

Minimal, copy-paste starting points for multimodal extraction. Each file lists
its own `pip install` line at the top.

| File | Approach | What it shows |
|------|----------|---------------|
| [vlm_extract.py](vlm_extract.py) | Vision-language model | Image + prompt -> structured JSON with Qwen2.5-VL |
| [docling_extract.py](docling_extract.py) | Document AI | PDF -> clean Markdown/JSON for downstream extraction |

For the full catalog of models, tools, and datasets, see the
[main list](../README.md).
